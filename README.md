# 2048 Game

2048 is a classic number-merging puzzle game built as a clean, structured game development project.  
This repository focuses on gameplay logic, system design, and maintainable code.

## Gameplay
- Move tiles in four directions
- Merge tiles with the same value
- Increase score with each merge
- Game ends when no moves are available

## Features
- Grid-based movement system
- Deterministic tile merge logic
- Score tracking
- Restart and game-over handling
- Clean separation of logic and UI

## Tech Stack
- Engine / Framework: Unity 
- Programming Language: C#
- Platform:  Web / Mobile
- Version Control: Git and GitHub

## Systems
- Tile spawning system
- Merge validation logic
- Game state management
- Input handling
